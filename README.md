# UI автотесты для Stellar Burgers

Проект содержит автотесты пользовательского интерфейса сайта  
**Stellar Burgers**: https://stellarburgers.education-services.ru

Тесты написаны с использованием **Python + Selenium + Pytest**.

---

# Покрываемый функционал

# Регистрация
- Успешная регистрация с валидными данными
- Ошибка при вводе пароля короче 6 символов

# Авторизация
- Вход через кнопку «Войти в аккаунт» на главной странице
- Вход через кнопку «Личный кабинет»
- Вход через форму регистрации
- Вход через форму восстановления пароля

# Личный кабинет
- Переход в личный кабинет
- Выход из аккаунта

# Конструктор бургеров
- Переходы к разделам:
  - Булки
  - Соусы
  - Начинки
