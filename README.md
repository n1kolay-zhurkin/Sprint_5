# Проект: автотесты для BooksCollector

В проекте реализованы автотесты для класса `BooksCollector`, который отвечает за управление книгами, их жанрами и списком избранного.

## Реализованные тесты

### add_new_book
- Проверка успешного добавления книги с валидным названием
- Проверка, что у добавленной книги жанр отсутствует

### set_book_genre / get_book_genre
- Проверка установки жанра существующей книге
- Проверка получения жанра по имени книги

### get_books_with_specific_genre
- Проверка получения списка книг с определённым жанром

### get_books_genre
- Проверка возврата словаря с книгами и жанрами

### get_books_for_children
- Проверка, что книги с возрастным рейтингом не попадают в список книг для детей

### add_book_in_favorites
- Проверка добавления книги в список избранного

### delete_book_from_favorites
- Проверка удаления книги из списка избранного

### get_list_of_favorites_books
- Проверка получения пустого списка избранных книг

## Дополнительно
- Используется параметризация тестов
- В некоторых тестах задействовано сразу несколько методов класса
- Все тесты успешно запускаются командой:

```bash
pytest -v tests.py
